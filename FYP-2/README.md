## Planned Features

### Push Notifications (FCM)
- Campaign announcements to volunteers
- Donation confirmation alerts
- New campaign availability notifications

### QR-Based Volunteer Attendance
- Admin generates QR code per campaign event
- Volunteer scans to mark attendance
- Real-time attendance tracking dashboard

### Volunteer-Campaign Matching Algorithm
- Rule-based matching (skills, location, availability)
- Recommendation engine for campaign suggestions
- Research paper material for academic contribution

### CSV/Excel Export
- Export donation data for NGO reporting
- Export volunteer participation records
- Campaign financial summaries
